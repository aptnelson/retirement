How This Was Made

From Excel spreadsheet to live web app using Claude Code
Back to Calculator

This retirement calculator started as an Excel spreadsheet (Personal Retirement Calculator.xlsm) with VBA macros, formulas, and charts. It was converted into the web app you see today using Claude Code, an AI programming tool made by Anthropic.

This page explains the process so you can do the same kind of thing yourself.

What is Claude Code?

Claude Code is a command-line tool that lets you have a conversation with an AI (Claude) that can read your files, write code, run commands, and build things for you. Think of it as a very capable programmer sitting next to you at your computer.

You talk to it in plain English. It reads your files, asks clarifying questions, writes code, and tests it. You stay in control the whole time — it asks for permission before doing anything significant.

How is this different from ChatGPT?

ChatGPT runs in a browser and can only see what you paste into the chat window. Claude Code runs on your computer — it can directly read your spreadsheet, create files, start a web server, and deploy a website. It's the difference between texting a contractor photos vs. having them walk through your house.

How the conversion happened

The whole process took about one conversation. Here's roughly what happened:

  1. Analyze the spreadsheet
    Claude Code opened the .xlsm file, read every sheet, extracted all formulas, identified the VBA macro, and mapped out how everything connects: which cells are inputs, which are calculations, what the charts display, and how the Monte Carlo simulation works.
  2. Create a project plan
    It wrote a CLAUDE.md file — a detailed specification capturing every formula, parameter, chart definition, and edge case. This acts as the "blueprint" for the project, and Claude Code reads it automatically in future sessions.
  3. Build the web app
    It created six files (HTML, CSS, and JavaScript) implementing all the math, three interactive charts, and a responsive layout. No frameworks, no build tools — just files that run directly in a browser.
  4. Verify the math
    It extracted actual numbers from the spreadsheet and compared them against its own calculations, verifying that the sensitivity analysis balances match to the penny (e.g. $864,157.43 at age 60, $922,139.70 at age 61).
  5. Fix discrepancies
    The spot-check revealed two issues: the expected return override was being applied incorrectly (replacing the whole blended return instead of just the stock return), and the Portfolio Decomposition chart was using smooth deterministic returns instead of the jagged stochastic returns the spreadsheet uses. Both were fixed.
  6. Deploy
    It created a GitHub repository and deployed the site to GitHub Pages, making it available at a public URL.

What's in each file

FileWhat it does
index.htmlThe page structure — all the input controls, chart areas, and layout
style.cssVisual styling — colors, spacing, responsive layout for mobile
calc.jsAll the math — the FV function, asset allocation glide path, projections, percentile calculations. This is the "brain" that replicates your spreadsheet formulas.
worker.jsRuns the Monte Carlo simulation (1,000 trials) in a background thread so the page stays responsive while it computes
charts.jsCreates and updates the three Chart.js charts
main.jsConnects everything — reads your inputs, runs the calculations, updates the charts
CLAUDE.mdThe project spec — every formula, parameter, and design decision. Claude Code reads this automatically when it works on this project.

How to make changes yourself

Install Claude Code

You need Node.js installed first. Then open a terminal and run:

npm install -g @anthropic-ai/claude-code

You'll need an Anthropic API key or a Claude subscription. The setup process will walk you through it.

Start a session

Open a terminal, navigate to the project folder, and type:

cd ~/path/to/retirement-calculator
claude

That's it. You're now in a conversation with Claude, and it can see all the project files.

Ask for changes in plain English

Just describe what you want. Some examples:

# Change a default value
"Change the default retirement age from 60 to 65"

# Add a feature
"Add a field for employer match percentage, where the employer
matches 50% of contributions up to 6% of salary"

# Fix something
"The withdrawal calculation should use 4% as the default,
not 10%"

# Understand something
"Explain how the asset allocation glide path works in calc.js"

# Change the look
"Make the charts taller and use a dark theme"

Claude Code will read the relevant files, make the changes, and explain what it did. It asks for your approval before modifying files.

Tip: the CLAUDE.md file is your secret weapon

Claude Code reads CLAUDE.md at the start of every session. It contains all the formulas and design decisions. If you want to change how something fundamental works (like adding a new asset class), start by asking Claude to update the spec in CLAUDE.md first, then implement the change. This keeps everything consistent.

Deploy updates

After making changes, just tell Claude Code:

"Commit these changes and push to GitHub"

GitHub Pages will automatically update within a minute or two.

Key things to know

  • You don't need to know how to code. Claude Code writes and modifies all the code. You just need to describe what you want clearly and verify the results make sense.
  • You stay in control. It asks before changing files, running commands, or deploying. You can say no to anything.
  • It makes mistakes sometimes. Just like any tool, it can get things wrong. That's why we spot-checked the math against the spreadsheet. When you see something off, just tell it — "that number looks wrong, it should be X" — and it will investigate and fix it.
  • It remembers context within a session but starts fresh each time you run claude. The CLAUDE.md file is how it "remembers" the project across sessions.
  • The spreadsheet is still the source of truth. If you find a discrepancy between the web app and the spreadsheet, the spreadsheet wins. Tell Claude Code to fix it.
